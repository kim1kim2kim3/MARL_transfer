Online multi-agent reinforcement learning에서는 새로운 cooperative task를 처음부터 학습하는 데 많은 interaction cost가 필요하므로, 이전 task에서 얻은 지식이나 행동 전략을 재사용하려는 시도가 자연스럽게 등장한다. 그러나 multi-agent setting에서 재사용되는 지식은 단일 agent의 행동 정책만이 아니라, 여러 agent 사이에서 형성된 coordination pattern까지 포함한다. 예를 들어 함께 이동하기, 특정 agent를 따라가기, 병목 지점에서 기다리기, 역할을 고정하는 행동은 어떤 source task에서는 효과적인 협동 전략일 수 있지만, target task의 구조가 달라지면 오히려 부적절한 inductive bias로 작용할 수 있다.

이 문제는 cooperative MARL에서 특히 중요하다. 하나의 coordination pattern은 개별 agent의 action 선택뿐 아니라 다른 agent의 탐색 경로, 역할 분담, joint behavior 전체에 영향을 주기 때문이다. 따라서 source task에서 성공적이었던 협동 습관이 target task에서도 유용하다고 가정하면, target task가 요구하는 새로운 협동 방식의 발견이 지연되거나, agents가 잘못된 역할 구조에 고착되거나, 초기 학습 성능이 scratch learning보다 나빠지는 negative transfer가 발생할 수 있다.
